# We escape using backslash - \
\b -one backspace character       
\n -New line         
\t -Tab
\r -Carriage return 
\o -One ASCII nul character  

`String str = 'My team\'s name is \'SFDCFacts Academy\'. \n We work on apex.';`
`System.debug(str);` // My team's name is 'SFDCFacts Academy'. 
We work on apex.
